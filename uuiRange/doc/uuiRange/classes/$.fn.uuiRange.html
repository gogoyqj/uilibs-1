$.fn.uuiRange Class

Defined in: js/uuiRange.js:8
Module: uuiRange

实现html 5的绑定input@type=range，支持多拖动头，rely on uuiDrag

Constructor

$.fn.uuiRange

(
  • options
)

Defined in js/uuiRange.js:8

Parameters:

  • options Object

    配置.

    • range Array

      起始区间，默认为 0 - 100.

    • duration Array

      选中区间或者点，长度为1或者2，默认为[0]，取值区间应该在range指定的范围内.

    • container String | Dom

      容器，默认为绑定ui组件的元素.

    • direction String

      range的防线，h为水平，v为竖直，默认水平.

    • enable Boolean

      使可用

    • callback Function

      设定值后执行回调,传递参数{Array}选中的range.

Example:

$('.uuiRange').uuiRange({enable:0})
